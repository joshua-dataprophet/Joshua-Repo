This repository contains a Flask web application that demonstrated how to set up, run, and test a web service using Python and Flask.

## Project Details:
- Listens on all interfaces (0.0.0.0)
- Runs on port 8000
- Exposes a single route /
- Returns the message: "Hello World!" when run

How to Run:
Make sure Python and Flask are installed.
Run python app.py in the project directory.
Open a browser at http://localhost:8000/ to see the message.
This app is a simple starting point for learning Flask and experimenting with web routes.
